# TA_PROGDAS_Muhammad-Habib_21120125130077_Spotipu-Program-Music-Player-dan-Manajer
  Spotipu adalah suatu program pemutar musik desktop yang menggunakan bahasa Python yang dirancang untuk memutar file Mp3 lokal secara offline. Menggunakan library berupa tkinter dan pygame untuk mesin suara dalam program tersebut, aplikasi ini mampu untuk memindai (scan) otomatis seluruh file Mp3 di folder komputer dan menampilkannya dalam daftar library tanpa perlu input manual. 
  
  Secara teknis, program ini menerapkan konsep Pemrograman Berorientasi Objek (OOP) dan struktur data yang efisien. Salah satu fitur uniknya adalah penggunaan struktur data Stack (Tumpukan) untuk menangani "Riwayat Pemutaran", yang memastikan lagu terakhir yang diputar selalu berada di urutan teratas untuk akses cepat. Selain itu, sistem ini menggunakan dictionary untuk mengelola basis data pengguna, memungkinkan sistem login sederhana di mana setiap pengguna memiliki data playlist pribadi yang terpisah satu sama lain.
  
  Dari sisi fitur manajemen, Spotipu tidak hanya sekadar memutar lagu, tetapi juga bertindak sebagai manajer musik pribadi. Pengguna dapat membuat akun, menyusun berbagai playlist sesuai suasana hati, menambahkan lagu dari library ke playlist tersebut, serta menghapus lagu dari playlist jika sudah tidak diinginkan. Aplikasi ini juga dilengkapi kontrol pemutaran lengkap (Play, Pause, Shuffle, Next/Prev) serta progress bar visual yang bergerak secara real-time mengikuti durasi lagu.
